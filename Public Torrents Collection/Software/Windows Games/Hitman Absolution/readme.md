## Hitman: Absolution System Requirements (Minimum)
- CPU: True dual core CPU (Intel, AMD)
- CPU SPEED: Info
- RAM: 4 GB
- VIDEO CARD: Nvidia Geforce 8600 512 Mb RAM, or AMD equivalent (Radeon HD 4650)
- DEDICATED VIDEO RAM: 512 MB
- PIXEL SHADER: 4.0
- VERTEX SHADER: 4.0
- OS: Windows Vista, 7
- SOUND CARD: Yes
